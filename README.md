# homebrew-mac-mic-mute

This repository is a Homebrew tap that publishes the `mac-mic-mute` cask for [Mac Mic Mute](https://github.com/jcleigh/mac-mic-mute).

## Install with brew

```bash
brew tap jcleigh/mac-mic-mute
brew install --cask jcleigh/mac-mic-mute/mac-mic-mute
```

## Upgrade

```bash
brew update
brew upgrade --cask jcleigh/mac-mic-mute/mac-mic-mute
```
